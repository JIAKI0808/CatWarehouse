# CatWarehouse README.md 编写计划

## 项目概述
CatWarehouse 是一个库存记录管理软件，支持 Windows 和 Android 双平台。

## 技术栈
- **后端 (CwServer)**: Python (FastAPI/Flask)
- **前端 (Cw_WebUi)**: Vue.js
- **Android**: WebView 套壳方案
- **语言**: 中英文双语
